# SpringBoot-JPA
- JPA와 H2 database를 활용하여 웹 애플리케이션 개발   
- 간단한 회원관리 및 상품주문 사이트
- 개발순서
```
1. 엔티티 클래스 개발
2. 도메인 개발 (회원,상품,주문) -> 리포지토리 & 서비스 
3. 웹 개발 (Form & Controller)
4. API 개발 
5. 성능 최적화
```
